@arcblock/mcrypto

Forge mcrypto implementation for javascript. Just a wrapper around existing javascript crypto libraries, implementation details can be found:

  • Signer
    • Ed25519 Ed25519Signer
    • Secp256k1 Secp256k1Signer
  • Hasher
    • SHA2 Sha2Hasher
    • SHA3 Sha3Hasher
    • KECCAK KecaakHasher
Source:
  • mcrypto/lib/index.js, line 1
Example
yarn add @arcblock/mcrypto

Example
const { Signer } = require('@arcblock/mcrypto');
const message = 'some message to sign';

// Use Signer directly
const keyPair = Signer.Ed25519.genKeyPair();
const signature = Signer.Ed25519.sign(message, keyPair.secretKey);
const result = Signer.Ed25519.verify(message, signature, keyPair.publicKey);
assert.ok(result);
